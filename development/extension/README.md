# Extension

## Preparation

- Create `.env` from `.env.example`
- Update `.env` with key from <https://openexchangerates.org/account/app-ids>

## Development

| Action            | Script          |
| ----------------- | --------------- |
| Install modules   | `npm ci`        |
| Start dev mode    | `npm run start` |
| Build application | `npm run build` |
